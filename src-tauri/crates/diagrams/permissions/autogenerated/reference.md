## Default Permission



#### This default permission set includes the following:

- `allow-subscribe-diagram`
- `allow-unsubscribe-diagram`
- `allow-update-feeders`
- `allow-update-events`
- `allow-event-open-breaker`
- `allow-event-close-breaker`
